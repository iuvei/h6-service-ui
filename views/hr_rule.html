  1. 如果客戶懷疑自己的資料被盜用，應立即通知本公司，並更改詳細資料，以前的使用者名稱及密碼將全部無效。
  2. 客戶有責任確保自己的帳戶及登入資料的保密性。以使用者名稱及密碼進行的 任何網上投注將被視為有效。
  3. 公佈賠率時出現的任何打字錯誤或非故意人為失誤，本公司保留改正錯誤和按 正確賠率結算投注的權力。您居住所在地的法律有可能規定網路博奕不合法；若此情況屬實，本公司將 不會批准您使用付帳卡進行交易。

  5. 一旦投注被接受，則不得取消或修改。
  6. 所有號碼賠率將不時浮動，派彩時的賠率將以確認投注時之賠率為准。
  7. 每注最高投注金額按不同[場次]及[投注項目]及[會員帳號]設定浮動。如投注 金額超過上述設定，本公司有權取消超過之投注金額。
  8. 所有投注都必須在開獎前時間內進行否則投注無效。

六合彩規則說明

  • 所有投注派彩彩金皆含本金。
  • 兩面（指大、小、單、雙）。

  • 特別號碼 (簡稱: 特碼)

    香港六合彩公司每期開出的最後一號碼為『特碼』。

  • 特碼大小

    特小：開出之特碼，(01 ~ 24) 小於 或 等於 24。
    特大：開出之特碼，(25 ~ 48) 大於 或 等於 25。
    和局：特碼為 49 時。

  • 特碼家禽野獸

    家禽：開出之特碼生肖為豬,羊,狗,馬,雞,牛。
    野獸：開出之特碼生肖為鼠,猴,龍,兔,虎,蛇。
    和局：特碼為 49 時 。

  • 特碼單雙

    特雙：特碼為『雙數』，如 18、20、34、42。
    特單：特碼為『單數』，如 01、11、35、47。
    和局：特碼為 49 時。

  • 特碼合數單雙

    合雙：指開出特碼的個位數加上十位數之和為『雙數』，如 02、11、33、44。
    合單：指開出特碼的個位數加上十位數之和為『單數』，如 01、14、36、47。
    和局：特碼為 49 時。

  • 正碼

    香港六合彩公司每期開出的前面六個號碼為正碼。下注號碼如在六個正碼 號碼裏為中獎

  • 總和大小

    總和大：所有七個開獎號碼的分數總和 大於 或 等於 175。如開獎號 碼為 01、09、27、29、33、42、45，分數總和 是 186，則為總和大。
    總和小：所有七個開獎號碼的分數總和 小於 或 等於 174。如開獎號 碼為 01、09、27、29、33、13、22，分數總和 是 144，則為總和小。

  • 總和單雙

    總和單：所有七個開獎號碼的分數總和是『單數』， 如分數總和是 133、197。
    總和雙：所有七個開獎號碼的分數總和是『雙數』， 如分數總和是 120、188。

  • 三全中

    選擇投注號碼每三個號碼為一組〈號碼需三個號以上，少於三個號碼不成立〉，對獎號是以正碼對獎〈1～6號不含特碼〉，如三個號碼都是在開獎號碼的正碼裏，視為中獎，其他情形視為不中獎。

  • 三中二

    中二：選擇投注號碼每三個號碼為一組〈號碼需三個號以上，少於三個號碼不成立〉，如中的二個號碼都是在開獎號碼的正碼裏，為三中二之中二視為中獎。
    中三：選擇投注號碼每三個號碼為一組〈號碼需三個號以上，少於三個號碼不成立〉，如中的三個號碼都是在開獎號碼的正碼裏，為三中二之中三視為中獎。
    除中二、中三外其他情形視為不中獎。
    例：11，17，18 為一組投注號碼，開獎號碼中有11，17兩個正碼〈沒有18號碼〉，為三中二中獎，賠率為三中二之中二的賠率，如果11，17，18三個號碼都是開獎之正碼，視為三中二之中三，賠率為三中二之中三的賠率，如只中一個號或是沒中任何號碼，此情形視為不中獎。

  • 二全中

    選擇投注號碼每二個號碼為一組〈號碼需二個號以上，少於二個號碼不成立〉，如二個號碼都是在開獎號碼的正碼，視為中獎，其他情形視為不中獎。
    例：11，17二個號碼都是開在正碼裏，視為中獎，如果二個號碼一個中在正碼一個中特碼，此情形視為不中獎。

  • 二中特

    選擇投注號碼每二個號碼為一組〈號碼需二個號以上，少於二個號碼不成立〉，如二個號碼都是在開獎號碼的正碼，為二中特之中二，如果一個中在正碼一個中特碼為二中特之中特是視為中獎，其他情形視為不中獎。
    例：11，17二個號碼都是開在正碼裏，視為中獎，如果二個號碼一個中在正碼一個中特碼，為二中特之中特是視為中獎，其他情形視為不中獎。

  • 特串

    選擇投注號碼每二個號碼為一組〈號碼需二個號以上，少於二個號碼不成立〉，如二個號碼其中一個開在正碼，另一個開在特碼，視為中獎，其他情形視為不中獎。如兩個號碼都開在正碼裏視為不中獎。

  • 正碼1-6特

    正1特、正2特、正3特、正4特、正5特、正6特：指下注的正碼特與現場滾波開出之正碼其開獎順序及開獎號碼相同，視視中獎，如現場滾波第一個正碼開獎為49號，下注第一個正碼特為49則視為中獎，其他號碼視為不中獎。

  • 正碼 1-6

    如 正一碼 開出為 32 ( 正 一碼 為雙、大、綠波、合數單 ) 。
    如 正二碼 開出為 19 ( 正 二碼 為單、小、紅波、合數雙 ) 。
    如 正三碼 開出為 49 ( 正三碼 單雙、大小、合數單雙為和局，但色波為綠波 ) 。

    以每二個尾數為一注，每注的每個尾數中如果都沒有當期開獎的所有7個號碼中對應的尾數，則視為中獎。如果其中一個或多個尾數有當期開獎的所有7個號碼對應的尾數，則視為不中獎。例如：尾1、尾2，是一注，開獎的號碼是：2、3、4、5、6、7、8，那就是不中獎；再例如：尾1、尾2，是一注，開獎的號碼是：5、6、7、8、9、 10、20，那就是中獎。

  • 投注方法簡稱：

    拖頭：拖頭是多組的簡化叫法。

    如二中二之8拖18、28、13、25，即是四組：8-18、8-28、8-13、8-25；
    如三中二之8、18拖19、20、21，即是三組：8-18-19、8-18-20、8-18-21；
    如二肖連之鼠拖牛、兔、虎、龍，即是四組：鼠-牛、鼠-兔、鼠-虎、鼠-龍；
    如三肖連之鼠、牛拖兔、虎、龍，即是三組：鼠-牛-兔、鼠-牛-虎、鼠-牛-龍；
    如四肖連之鼠、牛、兔拖虎、龍，即是兩組：鼠-牛-兔-虎、鼠-牛-兔-龍。

    複式：複式是多個（號碼、生肖、尾數）的不完全重複組合。

    多注投注：可以連續選多組。

    對碰：對碰是兩組對連的叫法：

    如：二全中，龍碰馬，即是二十組：01,11、01,23、01,35、01,47、11,13、13,23、13,35、13,47、11,25、23,25、25,35、25,47、11,37 23,37、35,37、37,47、11,49、23,49、35,49、47,49
